# dotfiles

Some config files for various tools

## Installation

```shell
git clone git@github.com:itimky/dotfiles.git && make -C dotfiles install
```

## Usage

```shell
make help
```

## Devcontainer

Wire devcontainer config in any directory with:

```shell
dotfiles-wire-devcontainer
```

tmux conf found [here](https://github.com/JohnMurray/dotfiles)
